# BDA-a_Kamal_AARAB
Git répertoriant les différents TP web du module BDAa.

## Pré-requis
TP1/TP2:
Installer Node js
Lancer la commande : 'npm install' dans les deux repository 'TP1/TP2'

## Exécution des TP
TP1: 
- node ./script.js <nom du fichier datas>

TP2:
- node ./server.js <nom du fichier datas>
